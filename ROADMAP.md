## Ваши следующие действия

### Шаг 1: Скачать демку (5 минут)

1. Откройте https://www.hltv.org/matches
2. Выберите любой завершённый матч (например, недавний матч Navi или G2)
3. Прокрутите вниз до "GOTV Demo"
4. Скачайте и распакуйте архив
5. Переместите файл `.dem` в папку `demos/`

### Шаг 2: Запустить парсер

```bash
cd /root/repo
source venv/bin/activate

# Посмотреть что внутри демки
python scripts/02_explore_demo.py

# Извлечь все данные
python scripts/03_extract_data.py

# Посмотреть статистику
python scripts/04_analyze_data.py
```

### Шаг 3: Проверить результаты

После запуска в папке `data/` появятся:
- `{demo_name}.json` - все данные в одном файле
- `{demo_name}/` - папка с CSV файлами:
  - `kills.csv`
  - `damages.csv`
  - `rounds.csv`
  - `grenades.csv`
  - `bomb_events.csv`
  - `economy.csv`
  - `players.csv`
